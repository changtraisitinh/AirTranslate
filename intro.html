흐름 작업 화면 기능 활용

Mac에서 들리는 소리를 작게 띄우고 오래 기록합니다.

AirTranslate는 외부 마이크가 아니라 Mac 시스템 오디오를 듣습니다. 영상, 회의, 강의에서 나오는 음성을 실시간 기록으로 남기고, 번역과 플로팅 자막까지 한 흐름으로 이어줍니다.

제품 둘러보기 핵심 기능

기록은 이어지고, 화면은 흐트러지지 않습니다.

실시간 기록은 채팅처럼 흩어지지 않고 현재 세션 안에 쌓입니다. 길어진 기록은 패널 안에서 스크롤되고, 캡처를 멈추면 저장소에서 다시 열어 편집할 수 있습니다.

Mac 재생음 캡처

스크롤 패널 기록

원문과 번역은 고정된 패널 안에서 쌓이고, 긴 기록은 내부 스크롤로 읽습니다.

일반 텍스트 저장

원문+번역은 같은 기본 이름의 _original.txt, _translation.txt 두 파일로 나누어 남깁니다.

▤

묶음 저장소

분리된 원문과 번역 파일을 저장소 목록에서는 하나로 묶고, 상세 화면에서는 두 패널로 함께 보여줍니다.

⧉

복사와 글쓰기 도구

원문과 번역 패널을 복사하고, 저장된 텍스트는 Apple Intelligence Writing Tools로 교정하거나 다시 쓸 수 있습니다.

⌂

로컬 중심

앱 자체 서버, 계정, API 키, 광고 SDK, 분석 SDK 없이 macOS 프레임워크 중심으로 동작합니다.

영상, 회의, 강의를 놓치지 않게.

귀로 듣는 정보를 화면과 파일로 바꿔, 다시 읽고 정리할 수 있는 자료로 만듭니다.
